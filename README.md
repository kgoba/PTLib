# uLIB
AVR/STM32 Peripheral Library

## HAL

* GPIO
* UART
* I2C
* SPI
* TIM
* ADC
* DAC
* DMA

## Devices

* Silicon Labs SI4xxx
* Silicon Labs SI5351x
* Various I2C sensors:
    - Magnetometers
    - Barometers
    - Accelerometers
* I2C IO extender
* I2C EEPROM
* SPI FLASH 
* SPI SD/MMC
* LCD text display modules
* LCD TFT modules

## Utilities

* Data structures:
    - Circular buffers
    - Queues
    - Fixed buffer strings
* Text functions:
    - Formatting
    - Text streams
* Console I/O
* GPS NMEA parser
* PWM generator
* Scheduling/Delay
* Software I2C/UART/SPI
* FAT
